СТРУКТУРА ПАПОК ПРОЕКТОВ
========================

Отображается всё содержимое папки projects/ — без привязки к году/месяцу.
Добавляйте любые папки и файлы — они появятся в дереве автоматически.

Поддерживаемые файлы:
  .mp4           — видео
  *_breakdown.txt — разбор
  *_tools.sys    — инструменты
  .md            — markdown

Пример:
  projects/
    клиенты/
      сбербанк/
        реклама.mp4
        реклама_breakdown.txt
    пет-проекты/
      neon_shell.mp4
