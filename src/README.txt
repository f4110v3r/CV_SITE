СТРУКТУРА ПАПОК
===============

Фото и шоурил:
  public/photo.jpg  — фото в блоке «О себе» (или photo.png)
  public/showreel.mp4 — стартовое видео

Проекты:
Добавляйте любые папки и файлы — они появятся в дереве автоматически.

Поддерживаемые файлы:
  .mp4           — видео
  .url           — ссылка на видео (YouTube, Vimeo, Behance/Adobe и т.п.)
                  Создайте файл video.url с одной строкой — URL видео
  *_breakdown.txt — разбор
  *_tools.sys    — инструменты
  .md            — markdown

Пример:
  projects/
    клиенты/
      сбербанк/
        реклама.mp4
        реклама_breakdown.txt
    пет-проекты/
      neon_shell.mp4
